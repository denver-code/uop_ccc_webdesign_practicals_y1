Modules Description

Architecture and Operating Systems

This module provides an in-depth study of computer architecture and operating systems, focusing on the fundamentals of how hardware and software interact to manage resources efficiently.

  • Understanding hardware components
  • Exploring OS functionalities
  • Resource management techniques

Comp Tutorial Level 4

The COMP Tutorial module offers hands-on guidance to first-year students on essential skills required for success in computing, such as problem-solving and programming best practices.

"Success in computing starts with mastering foundational skills and techniques."

Core Computing Concepts

Core Computing Concepts introduces students to key ideas in computer science, including algorithms, data structures, and computational thinking.

Database Systems Development

This module explores the principles of database design and development, with an emphasis on relational databases and SQL.

  • Designing relational databases
  • Learning SQL for database queries
  • Managing database structures

Networks

In this module, students will learn the basics of networking, including networking models, protocols, and the functioning of different network devices.

Programming
